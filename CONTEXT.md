# Mark Rover

Mark Rover is a local markdown reader focused on fast file opening, safe rendering, and comfortable long-form reading.

## Language

**Document**:
A local markdown file opened for reading in Mark Rover.
_Avoid_: Page, note, article

**Prototype Candidate**:
A self-contained package under `.prototype/` that tests a specific product or technical step.
_Avoid_: App version, experiment folder

**Corpus**:
A set of markdown documents used to test rendering, security, performance, and reader ergonomics.
_Avoid_: Fixtures, samples

**First Viewport Ready**:
The moment the initially visible portion of a document is painted and usable.
_Avoid_: Loaded, ready, first contentful paint

**External Link**:
A link from a document to a non-local web destination that requires user confirmation before opening.
_Avoid_: Web link, outbound link
